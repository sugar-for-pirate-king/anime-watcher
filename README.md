### Anime Watcher

Just simple app, alternative/another version of reddit-clone by ng-book.

#### Usage

1. Install node.js and angular
2. Clone this repo
3. `cd` to project root folder.
4. Run `ng serve`
5. Open browser and type url: `localhost:4200`
